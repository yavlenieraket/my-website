# Floating Background Images

Place background floating images here for the collision layer on the home page.

Use these names (or update `imageUrls` in `scripts.js`):
- `float-01.jpg`
- `float-02.jpg`
- `float-03.jpg`
- `float-04.jpg`
- `float-05.jpg`
- `float-06.jpg`

Recommendations:
- black/white or low-saturation edits
- 1200px+ on the long side
- JPG or WebP
